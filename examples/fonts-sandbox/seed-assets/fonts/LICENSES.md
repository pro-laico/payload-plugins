# Bundled sample fonts

These four fonts are bundled only as **seed/demo data** for the `fonts-only`
template. Each is an open-source Google font, distributed under the **SIL Open
Font License 1.1 (OFL)**. The `.woff2` files here are the Latin, 400-weight
static subsets, fetched from the [Fontsource](https://fontsource.org) CDN
(jsDelivr).

| File | Font | Family | Designer / Source | License |
| --- | --- | --- | --- | --- |
| `inter.woff2` | Inter | sans | Rasmus Andersson — <https://fonts.google.com/specimen/Inter> | OFL 1.1 |
| `lora.woff2` | Lora | serif | Cyreal — <https://fonts.google.com/specimen/Lora> | OFL 1.1 |
| `jetbrains-mono.woff2` | JetBrains Mono | mono | JetBrains — <https://fonts.google.com/specimen/JetBrains+Mono> | OFL 1.1 |
| `abril-fatface.woff2` | Abril Fatface | display | TypeTogether — <https://fonts.google.com/specimen/Abril+Fatface> | OFL 1.1 |

The full OFL text is available at <https://openfontlicense.org>. Replace these
with your own licensed fonts before shipping a real project.
