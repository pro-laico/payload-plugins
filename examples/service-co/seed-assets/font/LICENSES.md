# Bundled sample fonts

These fonts are bundled only as **seed/demo data**. Each is an open-source
font distributed under the **SIL Open Font License 1.1 (OFL)**. The static
`.woff2` files are Latin subsets from the [Fontsource](https://fontsource.org)
CDN (jsDelivr); the Inter variable files are from the official
[Inter release](https://rsms.me/inter/), and the Recursive variable file is a
Fontsource full-axes build of [Arrowtype Recursive](https://www.recursive.design/).
