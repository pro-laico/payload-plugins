Each plugin is published independently under the `@pro-laico/*` scope and shares one
lockstep version with the rest of the monorepo.

<Cards>
  <Card

  <Card

  <Card

    description="Custom fonts for Payload — a Font typeface collection that subsets uploads to served WOFF2s, a fontSet global, and an export endpoint + CLI for next/font/local."
  />
  <Card
    title="payload-icons"
    href="/docs/plugins/payload-icons"
    description="SVG icons for Payload — an Icon collection that optimizes and sanitizes SVGs on save (svgo + currentColor theming), a drop-in <Icon name> component that fetches and inlines the SVG, and native declarative seeding."
  />
  <Card
    title="payload-images"
